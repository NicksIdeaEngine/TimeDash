TimeDash
========

Change RescueTime.inc.new to RescueTime.inc and WakaTime.inc.new to WakaTime.inc

Add your API key to `@Resources\RescueTime.inc` and `@Resources\WakaTime.inc`

![Screenshot of TimeDash](screenshot.png)
